# **App Name**: Sebastian Brizuela - Portfolio

## Core Features:

- Profile Showcase: Display Sebastian's professional profile, including a concise summary, contact information (email, phone, LinkedIn, GitHub), and availability.
- Education Summary: Showcase Sebastian's academic background, listing his degree in progress at the University of Buenos Aires and his high school diploma.
- Technical Skills: A comprehensive list of programming languages, tools, and technologies Sebastian has experience with, enhancing SEO and demonstrating his expertise.
- Featured Projects: Highlights key projects from Sebastian's GitHub repositories with direct links to the code, providing tangible examples of his skills. Projects include MyFiubaGymBro, ReservasHoteles, Tetris, Snake, CriticasPeliculas, and Andypolis.
- Skills Matcher: Analyzes the job description provided by the user and highlights which skills and technologies match Sebastian's profile. Generates a summary to highlight his fit for the role.

## Style Guidelines:

- Primary color: HSL 214, 92%, 48% (Hex: #0D73D1), a vibrant, trustworthy blue reflecting competence and innovation.
- Background color: HSL 214, 20%, 95% (Hex: #F0F4F8), a very light blue to keep the focus on content.
- Accent color: HSL 184, 57%, 44% (Hex: #3E9A8A), a teal/cyan to highlight skills and links to Github repos.
- Body text: 'PT Sans', a humanist sans-serif, should be used throughout the app to be easy to read and convey a friendly, modern aesthetic.
- Employ a clean, modern, single-page layout to facilitate easy navigation and a straightforward user experience. Implement clear sections for each category of information.
- Utilize simple, professional icons to represent skills, tools, and links to external profiles, enhancing the visual appeal and user experience.
- Incorporate subtle hover effects on project cards and skill listings to provide visual feedback and encourage interaction, making the site feel dynamic and responsive.
